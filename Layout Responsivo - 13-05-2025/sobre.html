Bootstrap O que é?

Bootstrap é um framework front-end gratuito e de código aberto, criado originalmente por desenvolvedores do Twitter em 2011. Ele facilita o desenvolvimento de interfaces modernas, responsivas e compatíveis com diferentes dispositivos.

Como o Bootstrap funciona?

  • HTML: Estrutura básica da página
  • CSS: Estiliza elementos com classes prontas
  • JavaScript: Adiciona interatividade como modais, carrosséis e dropdowns
